# 🌟 Silas' Ultimate Hub

A comprehensive, accessibility-focused digital knowledge repository designed specifically for neurodivergent users.

## 🎯 Project Overview

Silas' Ultimate Hub is a React-based web application that serves as a self-contained therapeutic portal, featuring curated content across literature, music, entertainment, games, and educational resources.

## ✨ Features

- **🧠 Neurodivergent Support**: ADHD, autism, and anxiety accommodations
- **♿ Accessibility-First**: WCAG 2.1 AA compliant design
- **📱 Mobile-First PWA**: Responsive design with offline capabilities
- **🎨 Theme System**: Light/dark mode, high contrast, reduced motion
- **📚 Rich Content**: 18 comprehensive databases with 9,247+ lines of curated content
- **🔍 Self-Contained**: No external API dependencies

## 🗂️ Content Databases

- **Literature**: Classic & Modern collections
- **Music**: Rush, Dylan, Piano, Music Theory
- **Entertainment**: Studio Ghibli, Star Trek
- **Games**: Indie & Comprehensive collections
- **Poetry**: World poetry collections
- **Resources**: Neurodivergent support, Science, Arts, Culture

## 🛠️ Tech Stack

- **React**: 19.1.0
- **React Router**: 7.6.3
- **Tailwind CSS**: 4.1.11
- **Vite**: 7.0.2
- **Jest**: 29.6.2

## 🚀 Quick Start

```bash
# Install dependencies
npm install --legacy-peer-deps

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```

## 🌐 Deployment

This project is optimized for static hosting platforms like Netlify, Vercel, or GitHub Pages.

### Netlify Deployment

1. Build the project: `npm run build`
2. Deploy the `dist` folder to Netlify
3. Configure redirects for React Router (included in build)

## 🧪 Testing

```bash
npm test              # Run all tests
npm run test:watch    # Run tests in watch mode
npm run test:coverage # Run with coverage report
```
